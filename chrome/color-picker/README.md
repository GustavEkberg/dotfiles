# Local Color Picker

Buildless Chrome extension for picking a visible pixel color and copying its RGB value.

## Install

1. Open `chrome://extensions`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select this folder: `chrome/color-picker`.

## Notes

- Uses Chrome's built-in `EyeDropper` API.
- Outputs `rgb(r, g, b)` and hex.
- Copy button copies RGB only.
